## 🐰 <br/> 🕳️

A better text editor for keeping track of and documenting work.

### Setup

	npm install

### Running

	npm run start

### Formatting code

	npm run format

### Other commands

#### Build CSS only

When developing, use

	npm run css:watch

which will build and watch for changes.

To build CSS for prod, run

	npm run css:build
